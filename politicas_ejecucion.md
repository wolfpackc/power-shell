# Políticas de ejecución

No se consideren al mismo nivel que los antivirus, aunque en mi opinion si que es una "barrera" de seguridad extra que impide que se ejecuten scripts.
Para un hacker existen varias opciones para poder evadir este tipo de seguridad. La solución mas simple consiste en no ejecutar un archivo, si no solamente texto

Al ejecutar solo texto, las politicas de ejecución no se entrometen ya que solo se activan con archivos y nunca con texto.
